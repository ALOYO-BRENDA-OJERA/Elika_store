# Elika Store

A custom-made e-commerce application built with React, Vite, Tailwind CSS, and shadcn-ui.

## Features

- Product browsing and filtering
- Shopping cart functionality
- Admin dashboard for managing products and categories
- Responsive design

## Tech Stack

- **Frontend**: React, TypeScript, Vite
- **UI Components**: shadcn-ui, Tailwind CSS
- **State Management**: React Context, TanStack Query
- **Icons**: Lucide React

## Getting Started

### Prerequisites

- Node.js (v18 or later)
- npm or bun

### Installation

1. Clone the repository:
   git clone <repository-url>
   cd elika-store

2. Install dependencies:
   
pm install

3. Start the development server:
   
pm run dev

## Development

- \
pm run dev\: Starts the development server.
- \
pm run build\: Builds the application for production.
- \
pm run lint\: Runs ESLint for code quality checks.
- \
pm run test\: Runs tests using Vitest.
